Webrtc Basics
FEBRUARY 23, 2020
Webrtc is a real time communication over the web.Here the connection is established between peers so that the peers can send media as well as data directly without an intermediate server.But webrtc require and intermediate server just before establishing the connections. This servers are known as signaling serversl
Kurento Media Server
FEBRUARY 23, 2020
We can establish webrtc communication in two ways. one is without a media server and the other is with media server. In the second one a media server plays important role in between two webrtc peers or users. Media server recieve media from one peer and send this media to another peer. One of the major free and open source webrtc media server is kurento. The advantages of media server based webrtc applications are recording, transcoding , scalability etc
Intel Media Server
FEBRUARY 24, 2020
Intel webrtc collboration suit is a webrtc media server. Which is mainly used for MCU based webrtc applications. It support clustering of different media server components.
MCU & SFU
FEBRUARY 24, 2020
In webrtc,video communication can be established in two ways , SFU and MCU.In SFU individual streams are played separately in the receiver side but in MCU individual video streams are combained to a single stream and this single stream plays in receiver side.
